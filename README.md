# SnapDoc

SnapDoc is a simple and lightweight mobile application that converts images into PDF documents. The project is built using Python with the Flet framework for the user interface and the Pillow library for image processing. The goal of SnapDoc is to provide a minimal and accessible tool that allows users to quickly transform images such as photos, scanned documents, notes, or screenshots into a single PDF file directly from their device.

Many document scanning applications available today are heavy, cluttered with unnecessary features, or require subscriptions. SnapDoc was created with the idea that a document utility should be straightforward, fast, and open. Instead of adding complex scanning algorithms or unnecessary cloud integrations, the focus of this project is simplicity and transparency. The application allows users to select images, process them, and convert them into a PDF without complicated steps.

SnapDoc is also designed as an open source project. The intention is not only to build a useful tool but also to create a small collaborative project where developers can contribute improvements. Anyone is free to explore the code, suggest enhancements, optimize performance, or extend the functionality of the application. Because the project is written in Python and uses the Flet framework, it is relatively easy for developers to understand and modify.

## Purpose of the Project

The main purpose of SnapDoc is to provide a lightweight alternative for converting images into PDF documents. Many students, professionals, and everyday users often need to combine multiple images into a single PDF file for sharing, documentation, or record keeping. This application attempts to solve that problem in the simplest possible way.

Another purpose of the project is educational. SnapDoc demonstrates how Python can be used to build cross platform applications with graphical interfaces. By combining Flet for UI development and Pillow for image manipulation, the project serves as an example of how simple tools can be combined to create practical applications.

## Features

SnapDoc currently provides a set of basic but useful features that allow users to quickly convert images into PDF files.

- Convert multiple images into a single PDF document
- Select images directly from the device
- Simple and minimal user interface
- Lightweight design focused on usability
- Image processing handled using Pillow
- Built entirely with Python using the Flet framework
- Open source and available for community contributions

The application is intentionally minimal. The design philosophy is to keep the core workflow simple: select images, process them, and generate a PDF file.

## Technology Stack

The project is built using a small but powerful set of technologies.

Python is used as the primary programming language. It allows rapid development and makes the project easy for other developers to understand and contribute to.

Flet is used to build the graphical user interface. Flet allows developers to create cross platform applications using Python without needing to write separate frontend code.

Pillow is used for image processing tasks. It handles loading images, manipulating them, and preparing them for PDF generation.

Together these tools make it possible to build a fully functional application while keeping the codebase relatively small and easy to maintain.

## Current Status

SnapDoc is currently a working project but still under development. The application performs its core task of converting images to PDF, but there are areas where improvements can be made.

Some users may notice that the application can feel slightly slow or laggy in certain situations. This may happen when processing large images or multiple files. Performance optimization is an important future goal for the project, and contributions that improve speed or efficiency are highly encouraged.
